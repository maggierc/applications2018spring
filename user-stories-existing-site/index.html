User Stories: Kelly the Leisure Rider

  • As Kelly, I want a multifunctional bike so that I can bring it to diverse places, like beaches and parks.
  • As Kelly, I want a bike with 1-3 speeds so that I don't have to worry about complicated biking experiences.
  • As Kelly, I want a bike in the middle/upper price range so that I get the exact style and look that I want.
  • As Kelly, I want a bike with storage attached so that I can bring lots of things with me when I explore.
  • As Kelly, I want a nicely-colored bike because I'll be riding with my friends and want to show it off.
  • As Kelly, I want a lightweight bike so that I can carry and travel with it easily.
  • As Kelly, I want a women's bike so that I can be comfortable while riding.
  • As Kelly, I want a bike that gives comfort over performance, because I'm just looking to ride with friends and explore.
  • As Kelly, I want a bike with good reviews, so that I can avoid ones that are suprisingly complicated or high-maintenance.
